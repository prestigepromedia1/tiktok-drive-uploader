Social Video Archiver

urls.txt → organized local videos

Batch download, rename, and optionally upload social media videos from any platform. Powered by yt-dlp. Local-only by default, with optional Google Drive upload.

Supported Platforms
TikTok Instagram Reels YouTube Shorts Twitter / X Facebook Reddit Vimeo Twitch + 1000 more via yt-dlp
PIPELINE

How It Works

01 INPUT

URL List

Plain text (one URL per line) or CSV file. Auto-detects url/link columns.

02 DOWNLOAD

yt-dlp Engine

Best-quality download with metadata extraction. Platform auto-detected from URL.

03 OUTPUT

Named Files

Renamed with template: {creator}_{platform}_{date}_{id}. Saved locally or uploaded to Drive.

FEATURES

Key Capabilities

Naming Templates

Tokens: {creator} {platform} {date} {id} {title}

CSV Auto-detect

Finds url/link/video columns automatically. No manual column mapping needed.

Local-First

Downloads to local folder by default. Drive upload is opt-in with --drive-folder.

Error Resilience

Failed downloads are logged and skipped. The batch continues without stopping.

Multi-Platform

Powered by yt-dlp. One tool handles every supported site with platform auto-detection.

Custom Output Dir

Use --output-dir to save videos anywhere. Defaults to ./archived/.

Quick Start

INSTALL
pip install yt-dlp
Core dependency. Google Drive deps only needed for upload.
BASIC USAGE
python social_video_archiver.py urls.txt
Downloads all URLs to ./archived/ with default naming.
CUSTOM NAMING
python social_video_archiver.py urls.txt \ --template "{platform}_{creator}_{date}"
WITH DRIVE UPLOAD
python social_video_archiver.py urls.csv \ --drive-folder FOLDER_ID
